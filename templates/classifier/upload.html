How it Works

This tool uses a Gradient Boosting model to classify TESS Objects of Interest (TOIs) as either Planet Candidates or False Positives.

Model Details

  • Algorithm: Gradient Boosting
  • F1-Score: ~88%
  • Features: 65 (from TESS TOI catalog)

Missing values are handled automatically via imputation and Winsorization.
